# getGameList Method
This method returns all available games.

You should always use the `id_hash` variable when opening game using the [getGame method](/api-methods/get-game).

Games that are eligible for the [addFreeRounds method](/api-methods/add-freerounds) are returned as `freerounds_supported: true`.

### Method Request
```json
{
  "api_login" : "d13b1ceb-6209-4163-b1cf-f304c28b81ec",
  "api_password": "dqRin2nfLosij2n88",
  "method": "getGameList",
  "show_additional": true, 
  "show_systems" : 0,
  "list_type" : 1,
  "currency": "USD"
}
```
